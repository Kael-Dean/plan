# แบบฟอร์มและ Layout สำหรับแผนการจัดการเรียนรู้
## หน่วยการเรียนรู้ที่ ๑ เรื่อง น้ำใส — ภาษาไทย ป.๒

> ไฟล์นี้ใช้เป็นข้อมูลอ้างอิงโครงสร้างเพื่อแปลงเป็น DOCX

---

## ๑. โครงสร้างหน้า (Page Layout)

| รายการ | ค่า |
|--------|-----|
| ขนาดกระดาษ | A4 (210 × 297 mm) |
| หัวเรื่องกลาง (Header) | "แผนการจัดการเรียนรู้ที่ ___" — ตัวหนา ตรงกลาง |
| ระยะขอบบน/ล่าง | ~2.5 cm |
| ระยะขอบซ้าย/ขวา | ~2.5 cm |
| ฟอนต์หลัก | TH SarabunPSK หรือ Cordia New ขนาด 16pt |
| ฟอนต์หัวข้อ | ตัวหนา (Bold) ขนาด 16pt |
| การจัดตำแหน่งข้อความ | ชิดซ้าย (justify) |

---

## ๒. โครงสร้าง Header ของแต่ละแผน

```
                    แผนการจัดการเรียนรู้ที่ [เลขไทย]          ← ตรงกลาง, ตัวหนา

กลุ่มสาระการเรียนรู้ภาษาไทย          ชั้นประถมศึกษาปีที่ ๒
หน่วยการเรียนรู้ที่ ๑ เรื่อง น้ำใส   เวลา ๑๕ ชั่วโมง
เรื่อง [ชื่อเรื่องของแผน]            เวลา [จำนวน] ชั่วโมง
```

- แถวที่ ๑: กลุ่มสาระฯ (ซ้าย) | ชั้น (ขวา) — จัดแบบ 2 คอลัมน์
- แถวที่ ๒: หน่วยการเรียนรู้ (ซ้าย) | เวลา ๑๕ ชม. (ขวา)
- แถวที่ ๓: เรื่อง (ซ้าย) | เวลาแผน (ขวา)
- เส้นคั่น: จุดประ (..........) ยาวตลอดหน้า

---

## ๓. โครงสร้างส่วนมาตรฐาน (Standard Section)

```
สาระที่ [เลข] [ชื่อสาระ]            ← ตัวหนา

มาตรฐาน [รหัส]    [คำอธิบายมาตรฐาน]   ← ตัวหนาเฉพาะรหัส, ย่อหน้าแบบ hanging indent
```

---

## ๔. โครงสร้างหัวข้อหลัก 8 หมวด

แต่ละแผนการสอนมีหัวข้อหลัก 8 หมวดตามลำดับ ดังนี้:

### หมวด ๑ — สาระสำคัญ
```
๑.  สาระสำคัญ                    ← หัวข้อหลัก, ตัวหนา

    ๑.  ความคิดรวบยอด / การเรียนรู้คำ    ← หัวข้อย่อย, ตัวหนา
            [ข้อความย่อหน้า]              ← ย่อหน้า 2 ระดับ

    ๒.  สมรรถนะสำคัญของผู้เรียน          ← หัวข้อย่อย, ตัวหนา
        -   [สมรรถนะ 1]
        -   [สมรรถนะ 2]
```

**การจัดย่อหน้า:** ระดับ 1 = ย่อหน้า 1 tab | ระดับ 2 = ย่อหน้า 2 tab

---

### หมวด ๒ — ตัวชี้วัด
```
๒.  ตัวชี้วัด                    ← หัวข้อหลัก, ตัวหนา

    มาตรฐาน [รหัส]  [คำอธิบาย]    ← ตัวหนาเฉพาะรหัส
    มาตรฐาน [รหัส]  [คำอธิบาย]    ← (กรณีมีหลายตัวชี้วัด)
```

---

### หมวด ๓ — จุดประสงค์การเรียนรู้
```
๓.  จุดประสงค์การเรียนรู้         ← หัวข้อหลัก, ตัวหนา

    ๑.  [จุดประสงค์ข้อ 1]
    ๒.  [จุดประสงค์ข้อ 2]
```

---

### หมวด ๔ — คุณลักษณะอันพึงประสงค์
```
๔.  คุณลักษณะอันพึงประสงค์        ← หัวข้อหลัก, ตัวหนา

    ๑.  รักความเป็นไทย
    ๒.  ใฝ่เรียนรู้
    ๓.  มีจิตสาธารณะ
    ๔.  มีวินัย
    ๕.  อยู่อย่างพอเพียง
```

*(เหมือนกันทุกแผน)*

---

### หมวด ๕ — สาระการเรียนรู้
```
๕.  สาระการเรียนรู้                ← หัวข้อหลัก, ตัวหนา

    -   [หัวข้อสาระ 1]
    -   [หัวข้อสาระ 2]
```

*(ใช้ขีดกลาง "-" เป็น bullet)*

---

### หมวด ๖ — กิจกรรมการเรียนรู้
```
๖.  กิจกรรมการเรียนรู้             ← หัวข้อหลัก, ตัวหนา

    ๑.  [กิจกรรม 1]
    ๒.  [กิจกรรม 2]
    ๓.  [กิจกรรม 3]
        -   [รายละเอียดย่อย]
        -   [รายละเอียดย่อย]
    ๔.  [กิจกรรม 4]
    ...
```

*(บางแผนมีกิจกรรมย่อยเป็น bullet "-" ใต้กิจกรรมหลัก)*

---

### หมวด ๗ — สื่อ/แหล่งการเรียนรู้
```
๗.  สื่อ / แหล่งการเรียนรู้        ← หัวข้อหลัก, ตัวหนา

    ๑.  [สื่อ 1]
    ๒.  [สื่อ 2]
    ๓.  [สื่อ 3]
```

---

### หมวด ๘ — วัดผลประเมินผล
```
๘.  วัดผลประเมินผล                 ← หัวข้อหลัก, ตัวหนา

    ๑.  วิธีการวัดและประเมินผล      ← หัวข้อย่อย, ตัวหนา
        -   [วิธีการ 1]
        -   [วิธีการ 2]

    ๒.  เครื่องมือการประเมิน        ← หัวข้อย่อย, ตัวหนา
        -   [เครื่องมือ 1]
        -   [เครื่องมือ 2]

    ๓.  เกณฑ์การประเมิน             ← หัวข้อย่อย, ตัวหนา
        -   สังเกตพฤติกรรมการอ่าน ผ่านเกณฑ์ร้อยละ ๘๐
        -   ประเมินการตรวจผลงานผ่านเกณฑ์ร้อยละ ๘๐
```

*(เกณฑ์เหมือนกันทุกแผน)*

## ๕. ระดับการย่อหน้า (Indentation Levels)

| ระดับ | คำอธิบาย | ตัวอย่าง |
|-------|----------|---------|
| L0 | หัวข้อหลัก (ชิดซ้าย) | `๑. สาระสำคัญ` |
| L1 | หัวข้อย่อยระดับ 1 (1 tab) | `    ๑. ความคิดรวบยอด` |
| L2 | เนื้อหาใต้หัวข้อย่อย (2 tab) | `        [ข้อความ]` |
| L1 | รายการแบบเลข (1 tab) | `    ๑. จุดประสงค์` |
| L1 | รายการแบบขีด (1 tab) | `    - รายการ` |
| L2 | รายการย่อยแบบขีด (2 tab) | `        - รายการย่อย` |

## ๖. รูปแบบตัวอักษร (Typography)

| องค์ประกอบ | รูปแบบ |
|-----------|--------|
| ชื่อแผน (บรรทัดแรก) | **ตัวหนา, ตรงกลาง** |
| ชื่อสาระ/มาตรฐาน | **ตัวหนา** |
| รหัสมาตรฐาน (เช่น ท ๑.๑) | **ตัวหนา** |
| หัวข้อหลัก (๑.–๘.) | **ตัวหนา** |
| หัวข้อย่อย | **ตัวหนา** |
| เนื้อหา/รายละเอียด | ปกติ |
| คำที่ต้องการเน้น | ปกติ (ไม่มีการเน้นพิเศษ) |

## ๗. สัญลักษณ์และการเลขที่ใช้

| ประเภท | สัญลักษณ์ที่ใช้ |
|--------|----------------|
| ลำดับหัวข้อหลัก | เลขไทย: ๑. ๒. ๓. ... |
| ลำดับย่อย | เลขไทย: ๑. ๒. ๓. ... |
| รายการ bullet | ขีด: - หรือ เส้นประ |
| เส้นคั่น | จุดประ: ............... |
| ตัวเลขในเนื้อหา | ใช้ทั้งเลขไทย (๑๕) และเลขอารบิก (5-6) |

## ๙. ความแตกต่างระหว่างแผน (Variations)

### หมวดที่มีความแตกต่าง:

| หมวด | แผน ๑-๒, ๓, ๕, ๗, ๙, ๑๐ | แผน ๔ | แผน ๖, ๘ |
|------|--------------------------|-------|----------|
| สาระ | สาระที่ ๑ การอ่าน | สาระที่ ๑ การอ่าน | **สาระที่ ๔ หลักภาษาไทย** |
| มาตรฐาน | ท ๑.๑ | ท ๑.๑ | **ท ๔.๑** |
| จำนวนตัวชี้วัด | ๑ ตัว | **๒ ตัว** (ป.๒/๔ และ ป.๒/๕) | ๑ ตัว |
| สมรรถนะ | สื่อสาร + ทักษะชีวิต | คิด + ทักษะชีวิต | คิด + ทักษะชีวิต |

### หมวดที่เหมือนกันทุกแผน:
- **คุณลักษณะอันพึงประสงค์**: เหมือนกันทุกแผน (5 ข้อ)
- **เกณฑ์การประเมิน**: เหมือนกันทุกแผน (ร้อยละ ๘๐ ทั้งคู่)
- **วิธีการวัด**: สังเกตพฤติกรรม + ตรวจผลงาน (เหมือนกันทุกแผน)
- **เครื่องมือ**: แบบสังเกต + แบบฝึก (เหมือนกันทุกแผน ยกเว้นแผน ๑ ที่ใช้ "แบบสังเกตพฤติกรรมการอ่านออกเสียงเป็นรายบุคคล")

## ๑๐. การแบ่งหน้าในไฟล์ DOCX

แต่ละแผนการสอน = **๑ ชุด** (Page Break ระหว่างแผน)

แต่ละแผนใช้พื้นที่ประมาณ **2 หน้า A4**:
- หน้า 1: Header + หมวด ๑–๕
- หน้า 2: หมวด ๖–๘
