# สเปกรูปแบบเอกสาร Word (แผนการจัดการเรียนรู้)

มาตรฐานหน้าตาของไฟล์ `.docx` ที่ `build_docx.mjs` สร้าง (อิงฟอร์มราชการไทย — ใช้ได้ทุกวิชา)

## หน้ากระดาษ + ฟอนต์
- ขนาด **A4** (11906 × 16838 twips), ระยะขอบ **2.5 cm** ทุกด้าน (1417 twips)
- ฟอนต์ `config.font` (default **TH SarabunPSK**) ขนาด `config.fontSizePt` (default **16 pt** = 32 half-points)
- เลขข้อใช้**เลขไทย** (๑ ๒ ๓ …) เสมอ

## โครงสร้างแต่ละแผน (เรียงตามนี้)
1. **หัวแผน** กลางหน้า: `แผนการจัดการเรียนรู้ที่ <เลขไทย>` (renumber ๑..N ตามลำดับใน `keepPlans`)
2. **Header ๒ คอลัมน์** (tab ซ้าย/ขวา) ๓ บรรทัด:
   - `กลุ่มสาระการเรียนรู้<subject>` | `<grade>`
   - `<unitTitle>` | `เวลา <unitTotalHours> ชั่วโมง`
   - `เรื่อง <ชื่อแผนจาก PDF>` | `เวลา <planHours> ชั่วโมง`
3. **เส้นจุดประ** คั่น (จุด ๑๒๐ ตัว กลางหน้า)
4. **สาระ/มาตรฐาน** (ถ้า PDF มี): บรรทัด `สาระที่ …` (ตัวหนา) + คำอธิบายมาตรฐาน (ย่อหน้า 0.6 cm)
5. **หัวข้อ ๑–๘** ดึงจาก PDF (ดู `pipeline.md` เรื่องการ extract):
   - ๑ สาระสำคัญ → `buildSection1` (แยก ความคิดรวบยอด / สมรรถนะสำคัญ)
   - ๒ ตัวชี้วัด, ๓ จุดประสงค์การเรียนรู้, ๕ สาระการเรียนรู้, ๖ กิจกรรมการเรียนรู้, ๗ สื่อ/แหล่งเรียนรู้, ๘ วัดผลประเมินผล → `buildGenericSection` (จัดย่อหน้าตามตำแหน่ง x)
   - ๔ คุณลักษณะอันพึงประสงค์ → `buildSection4` (รายการจาก `config.kulaksana`)
6. **หัวข้อ ๙ การวัดและประเมินผล** → `buildSection9` (ตาราง ๔ คอลัมน์: ทักษะ/วิธีการ/เครื่องมือ/เกณฑ์
   จาก `config.section9Rows`) + บรรทัด **เกณฑ์การตัดสิน/ระดับคุณภาพ** (จาก `config.gradingScale`)
7. **หัวข้อ ๑๐ กิจกรรมเสนอแนะ** + ลายเซ็นครูผู้สอน
8. **บันทึกหลังแผนการจัดการเรียนรู้** (ขึ้นหน้าใหม่) → ๑ ผลการสอน / ๒ ปัญหาและอุปสรรค / ๓ ข้อเสนอแนะ + ลายเซ็นผู้บันทึก
9. **ลายเซ็น ๒ ตำแหน่งล่าง**: หัวหน้ากลุ่มสาระ (`homeroomLabel`) | รองผู้อำนวยการ (`deputyLabel`)
10. **Page break** ระหว่างแผน (๑ แผน = ๑ ชุด)

## ลายเซ็น ๓ บล็อก (สำคัญ)
สร้างด้วย **ตารางไม่มีขอบ** (`borders: NONE`) + **`AlignmentType.CENTER`** ทุกย่อหน้าในเซลล์ + **`TableRow.cantSplit: true`**
- บน (หลังข้อ ๑๐): ๑ เซลล์ ๑๐๐% — `ลงชื่อ……ครูผู้สอน` / `(ชื่อผู้สอน)`
- กลาง (หลังบันทึกหลังสอน): ๑ เซลล์ ๑๐๐% — `ลงชื่อ……ผู้บันทึก` / `(ชื่อผู้สอน)` / `ครูผู้สอน`
- ล่าง: ๒ เซลล์ ๕๐/๕๐ — (ซ้าย) `ลงชื่อ…` / `()` / `<homeroomLabel>` ; (ขวา) `ลงชื่อ…` / `()` / `<deputyLabel>`

## การ normalize เวลา (DEFAULT)
ทุกแผนตั้ง **`planHours` = ๑ ชั่วโมง** และหัวหน่วย **`unitTotalHours` = ๑๐ ชั่วโมง** (override ค่าใน PDF เสมอ)
- ถ้า PDF มีแผน **มากกว่า ๑๐** → ยุบเหลือ ๑๐ โดยตั้ง `keepPlans` ให้เลือกเฉพาะ ๑๐ แผนหลัก
  (ปกติเก็บแผน ๑–๑๐ ที่ลงท้ายด้วย "ชวนคิด ชวนทำ" แล้วตัดแผนอ่านเสริมท้าย ๆ — ถามผู้ใช้ก่อนยุบ)
- เก็บแผนให้ครบ ไม่ตัด ยกเว้นต้องยุบให้พอดี ๑๐ หรือมีแผนซ้ำจริง

> รายละเอียดกฎจัดหน้า (keep-together) ดู `page-format.md`
