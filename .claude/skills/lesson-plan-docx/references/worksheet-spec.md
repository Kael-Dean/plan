# สเปกใบงาน (worksheets)

ใบงาน ๑ ใบต่อ ๑ แผน สร้างเป็น HTML (การ์ดสีน่ารักสำหรับเด็ก) แล้วแปลงเป็น PDF ด้วย Microsoft Edge headless
ตัวสร้างคือ `generate_worksheets.mjs` (template) — **คัดลอกไปไว้ใน `<unitDir>/แบบฝึกหัด/` แล้วเติม `WORKSHEETS` array**

## ที่มาของเนื้อหา (สำคัญที่สุด)
1. อ่าน **"๖. กิจกรรมการเรียนรู้"** ของแต่ละแผน (จาก `pdf_data.json` หรือ dump ดู `pipeline.md`)
2. อ่าน **เนื้อหาจริงจากรูปหน้าหนังสือเรียน** ในโฟลเดอร์ `config.imageFolder` (เช่น คำศัพท์, เรื่อง, บทร้องเล่น,
   ตัวอย่างไวยากรณ์) เพื่อให้ใบงานตรงเนื้อหาจริง ไม่ใช่เนื้อหาลอย ๆ
3. `topic` / `objective` ของใบงาน = จาก **"๓. จุดประสงค์การเรียนรู้"** ของแผนนั้น

## กฎเนื้อหา (Hard rules)
1. **ใบงานต้องเป็นแบบฝึก ไม่ใช่อ่านตามหนังสือ** — ห้ามกิจกรรมที่แค่ "ดูภาพ/อ่านคำในหนังสือ" (เด็กเปิดอ่านเองได้)
   ต้องเป็นกิจกรรมที่ต้อง **คิด/เขียน/ทำ**: เติมคำ โยงเส้น แยกประเภท แต่งประโยค ตอบคำถาม คัดคำ ฯลฯ
2. **ห้ามคำถามซ้ำ** — กิจกรรมที่ถามสิ่งเดียวกัน ไม่ว่าในใบเดียวกันหรือข้ามใบ ให้เก็บไว้เพียงที่เดียว
3. ใช้รูปจากโฟลเดอร์รูปหนังสือเรียนเท่านั้น (ถ้าจำเป็นต้องใช้รูป)
4. คะแนนเต็มใบละ ๒๐ (footer มีช่องคะแนน/ลายเซ็นผู้ปกครอง/ผู้ตรวจ)

## ตัวช่วยสร้างกิจกรรม (มีใน template แล้ว)
`card(color,num,title,body)` ห่อทุกกิจกรรม — สี: yellow / blue / green / pink / purple
- `spellActivity(words, example)` — ตารางแจกลูกสะกดคำ (เส้น ๓ ระดับแบบกระดาษคัด)
- `matchActivity([{w,m}])` — โยงเส้นจับคู่ (มีจุดเชื่อม ๒ ฝั่ง)
- `copyActivity(words)` — คัดคำตัวบรรจง
- `wordGridActivity(words)` — ตารางคำให้วงกลม/เลือก
- `songActivity(title, lines, hintWords)` — บทร้องเล่น/เพลงเติมคำ (ใช้ `<span class="blank"></span>`)
- `fillSentences(color, hint, sentences)` — เติมคำในประโยค (ใช้ `@` เป็นช่องว่าง)
- `catActivity(headers, rows, hint)` / `tableActivity(color, headers, rows)` — แยกประเภท/ตารางคำตอบ (`""` = ช่องเขียน)
- `qaBox(color, questions)` — ตอบคำถาม (เว้นบรรทัดเขียน)
- `panelsActivity([{label}])` — ช่องข้อคิด/นำไปใช้
- `drawBox(color, h, caption)` — กรอบวาดภาพ + บรรยาย
- `writeBox()` — บรรทัดเขียนยาว

## ตารางแจกลูกสะกดคำ — เส้น ๓ ระดับ
CSS `.spell td.line` ใช้ gradient ๓ เส้น (บนเข้ม + กลางอ่อน + ล่างเข้ม สูง 38px) แบบกระดาษคัดลายมือ ป.ต้น — มีใน CSS แล้ว

## แปลง HTML → PDF
Edge headless: `msedge --headless=new --no-margins --print-to-pdf="out.pdf" --print-to-pdf-no-header "file:///…html"`
- ตั้งชื่อ PDF: `ใบงานที่_<เลขไทย>_บทที่_<unitNo>_<unitName>.pdf` (ต้องตรงกับที่ `embed_worksheets.mjs` หา)
- ใบงานมักยาว ๑–๒ หน้า A4 (ปกติ) — embed รองรับหลายหน้า
